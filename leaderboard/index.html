How it works

You gain a random amount of money and experience every time you text something or execute a command. To prevent people from spamming, there is a cooldown, so you can only get experience and money once every few seconds.

When you level up, you get money rewards and role rewards if the server owner set them up. Also, the RPG is global, which means you have the same Level, Balance, etc. on every server which uses Twilight Sparkle.
